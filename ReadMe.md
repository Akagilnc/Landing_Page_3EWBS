## Technology stack

- Language: [TypeScript v5][4]
- Component engine: [WebCell v3][5]
- Component suite: [BootCell v2][6]
- PWA framework: [Workbox v7][7]
- Package bundler: [Parcel v2][8]
- CI / CD: GitHub [Actions][9] + [Pages][10]
